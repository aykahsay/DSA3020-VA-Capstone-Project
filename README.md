# Crop Yield Prediction

A capstone project that collects crop yield data from FAOSTAT, builds ML models to predict yields,
and deploys a simple Streamlit app.

Structure:
- data/: raw/ processed/ external/
- notebooks/: EDA & training notebooks
- src/: main project code (collection, preprocessing, training, evaluation)
- app/: Streamlit demo
- models/: saved model & scaler
